## 6. 启动顺序

建议启动顺序：

1. 先启动 MySQL
2. 再启动 Redis
3. 再启动后端 `RuoYi-Vue-fast`
4. 最后启动前端 `RuoYi-Vue3`

## 7. 后端启动

进入目录：

```powershell
cd E:\接单项目\若依框架\RuoYi-Vue-fast
```

常见启动方式：

```powershell
mvn spring-boot:run
```

或先打包再运行：

```powershell
mvn clean package -DskipTests
java -jar target\ruoyi.jar
```

后端默认端口：

- `8080`

接口根地址：

- `http://localhost:8080`

## 8. 前端启动

进入目录：

```powershell
cd E:\接单项目\若依框架\RuoYi-Vue3
```

安装依赖：

```powershell
npm install
```

开发启动：

```powershell
npm run dev
```

生产构建：

```powershell
npm run build:prod
```

前端启动后，按项目本地开发端口访问即可。
